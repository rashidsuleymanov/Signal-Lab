## Marketplace skills to connect (minimum 6)

Connect these marketplace skills in Cursor for this repo, and keep them enabled:

- **docker-expert**: faster iteration on `docker-compose.yml` and container debug
- **nestjs-best-practices**: idiomatic NestJS structure, pipes/filters/interceptors patterns
- **prisma-orm**: schema/migrations ergonomics, query patterns
- **shadcn-ui**: component patterns compatible with shadcn conventions
- **tailwind-design-system** (or Tailwind v4 / shadcn oriented): layout + tokens conventions
- **next-best-practices** (or Vercel React best practices): App Router patterns, env, data fetching

### Why custom skills still exist

Marketplace skills are generic. Custom skills encode **repo-specific** constraints:
- exact metric names and labels used in Signal Lab
- ports and verification walkthrough expectations
- paths and scripts for Prisma in this monorepo layout
